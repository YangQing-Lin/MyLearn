##### 用数组表示队列：

```c++
// hh 表示队头，tt 表示队尾、

// 在队尾插入元素，在队头弹出元素
int q[N], hh, tt = -1;

// 插入
q[++ tt];

// 弹出
q[hh ++];

// 判断队列是否为空
if (hh <= tt) not empty;
else empty;

// 取出队头元素
q[hh];
```
