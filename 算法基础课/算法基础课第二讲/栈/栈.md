##### 用数组模拟栈：

```c++
#include <iostream>

using  namespace std;

const int N = 100010;

int stk[N], tt;

// 插入
stk[++ tt] = x;

// 弹出
tt --;

// 判断栈是否为空
if (tt > 0) not empty
else empty
    
// 栈顶
stk[tt];
```
