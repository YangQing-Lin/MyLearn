#### 题目1：<a href="https://www.acwing.com/problem/content/797/">前缀和</a>

----------------

##### 思路：

1. 预处理前缀和的数组
2. 用公式求区间和

--------------------

##### 题解：

```c++
#include <iostream>

using namespace std;

const int N = 100010;

int n, m;
int a[N], s[N]; // s[N] 是前缀和数组

int main()
{
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i ++) scanf("%d", &a[i]);
    
    for (int i = 1; i <= n; i ++) s[i] = s[i - 1] + a[i]; // 前缀和的初始化
    
    while (m --)
    {
        int l, r;
        
        scanf("%d%d", &l, &r);
        printf("%d\n", s[r] - s[l - 1]);
    }
    
    return 0;
}
```
