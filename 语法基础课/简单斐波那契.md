##### 原题：<a href="https://www.acwing.com/problem/content/description/719/">简单斐波那契</a>

##### 题解：

```c++
#include <iostream>

using namespace std;

int main()
{
    int n;
    cin >> n;

    int a = 0, b = 1;

    for (int i = 0; i < n; i ++ )
    {
        cout << a << ' ';
        int c = a + b;
        a = b;
        b = c;
    }

    cout << endl;

    return 0;
}
```
