#### <a href="https://leetcode.cn/problems/smallest-even-multiple/">最小偶倍数</a>

-------------

```java
class Solution {
    public int smallestEvenMultiple(int n) {
        if (n < 2) {
            return 2;
        }
        if (n % 2 == 0) {
            return n;
        }

        int res = 0;
        for (int i = 1; i <= n; i ++) {
            if (n * i % 2 == 0) {
                res = n * i;
                break;
            }
        }
        return res;
    }
}
```
