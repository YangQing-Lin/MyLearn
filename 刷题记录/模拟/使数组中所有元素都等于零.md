#### <a href="https://leetcode.cn/problems/make-array-zero-by-subtracting-equal-amounts/">使数组中所有元素都等于零</a>

-------------

```java
class Solution {
    public int minimumOperations(int[] nums) {
        int n = nums.length;
        int res = 0;

        int sum = 0;
        for (int i = 0; i < n; i ++) {
            sum += nums[i];
        }
        
        while (sum != 0) {
            res ++;
            int min = Integer.MAX_VALUE;
            for (int i = 0; i < n; i ++) {
                if (nums[i] > 0) {
                    min = Math.min(min, nums[i]);
                }
            }
            
            for (int i = 0; i < n; i ++) {
                if (nums[i] != 0) {
                    nums[i] -= min;
                    sum -= min;
                }
            }
        }
        return res;
    }
}
```
