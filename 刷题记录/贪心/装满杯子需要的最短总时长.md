#### <a href="https://leetcode.cn/problems/minimum-amount-of-time-to-fill-cups/">装满杯子需要的最短总时长</a>

---------------------

每次找到杯子剩余容量最大的两个杯子装水

```java
class Solution {
    public int fillCups(int[] amount) {
        int res = 0;
        while (amount[0] + amount[1] + amount[2] != 0) {
            Arrays.sort(amount);
            amount[2] --;
            if (amount[1] != 0) {
                amount[1] --;
            }
            res ++;
        }
        return res;
    }
}
```
