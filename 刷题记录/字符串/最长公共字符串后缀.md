#### <a href="https://www.acwing.com/problem/content/781/">最长公共字符串后缀</a>

----------------

##### 主要思路：三重循环

![最长公共字符串后缀](C:\Users\冬黎\OneDrive\图片\语法基础课\最长公共字符串后缀.png)

- 第一重循环从最大的后缀长度开始递减
- 第二重循环枚举除了第一个字符串的其他字符串
- 第三重循环将第二重循环枚举的每个字符串与第一个字符串字符对比后缀，如果有字符不匹配，后缀长度减一

###### c++

```c++
#include <iostream>
#include <cstring>
#include <algorithm>

using namespace std;

const int N = 200;

int n;
string str[N];

int main() {
    while (cin >> n, n) {
        int len = 0;
        for (int i = 0; i < n; i ++) {
            cin >> str[i];
            int a = str[i].size();
            len = max(len, a);
        }
        
        while (len) {
            bool is_success = true;
            for (int i = 1; i < n; i ++) {
                bool is_same = true;
                for (int j = 1; j <= len; j ++) {
                    if (str[0][str[0].size() - j] != str[i][str[i].size() - j]) {
                        is_same = false;
                        break;
                    }
                }
                
                if (!is_same) {
                    is_success = false;
                    break;
                }
            }
            
            if (is_success) {
                break;
            }
            len --;
        }
        
        cout << str[0].substr(str[0].size() - len) << endl;
    }
    
    return 0;
}
```

###### java

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        while (true) {
            int n = sc.nextInt();
            if (n == 0) {
                break;
            }
            
            String[] str = new String[n];
            int len = 200;
            for (int i = 0; i < n; i ++) {
                str[i] = sc.next();
                int a = str[i].length();
                len = Math.min(len, a);
            }

            while (len != 0) {
                boolean is_success = true;
                for (int i = 1; i < n; i ++) {
                    boolean is_same = true;
                    for (int j = 1; j <= len; j ++) {
                        if (str[0].charAt(str[0].length() - j) != str[i].charAt(str[i].length() - j)) {
                            is_same = false;
                            break;
                        }
                    }

                    if (!is_same) {
                        is_success = false;
                        break;
                    }
                }

                if (is_success) {
                    break;
                }
                len --;
            }

            System.out.println(str[0].substring(str[0].length() - len));
        }
    }
}
```
